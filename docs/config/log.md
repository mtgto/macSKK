# ログ

設定画面の「ログ」タブにある機能です。

macSKKのアプリケーションログを表示します。辞書の読み込みエラーや設定ファイルの解析エラーなど、動作に関するメッセージを確認できます。

## 操作

**コピー**: ログ全体をクリップボードにコピーします。

## ログの見方

各行は以下の形式で表示されます。

```
[日時] [レベル] メッセージ
```

| レベル | 意味 |
|---|---|
| `info` | 通常の動作ログ |
| `error` | エラー（動作に支障をきたす可能性がある問題） |
| `fault` | 重大なエラー |

## 活用例

- 辞書ファイルが読み込まれない場合、`error` レベルのメッセージで原因を確認できます
- ローマ字かな変換ルールファイルに構文エラーがある場合、該当行番号がログに出力されます
