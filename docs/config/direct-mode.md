# 直接入力

設定画面の「直接入力」タブにある設定項目です。

直接入力を有効にしたアプリでは、macSKKによる日本語変換を行わずキー入力をそのままアプリに渡します。独自のIME機能を持つEmacs.appなどでの使用を想定しています。

## アプリの一覧

直接入力が有効なアプリがアイコン・名前とともに一覧表示されます。「Delete」ボタンで無効化できます。

## 直接入力の追加方法

直接入力させたいアプリを最前面に表示した状態で、**入力メニュー → "(アプリ名)で直接入力"** を選択します。

## 注意事項

- Bundle Identifier単位で記録されるため、アプリを移動しても設定は維持されます
- Bundle Identifierを持たない特殊なGUIアプリ（Android StudioのAndroidエミュレータなど）は設定できません
