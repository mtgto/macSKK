# ソフトウェアアップデート

設定画面の「ソフトウェアアップデート」タブにある設定項目です。

## 表示内容

- **現在のバージョン:** 現在インストールされているmacSKKのバージョン
- **最新バージョン** 最新リリースのバージョン（確認後に表示）
- 最新リリースのリリースノート

## 操作

**アップデートを確認…**

今すぐ最新バージョンを確認します。

**リリースページを開く**

GitHubのリリースページを開きます。

## 自動チェックの仕組み

12時間おきにバックグラウンドでGitHub ReleasesのAtomフィードを取得します。新バージョンが見つかった場合は通知センターで通知します。

macSKK本体はApp Sandboxによりインターネット通信ができないため、バージョンチェックはXPC経由の外部プロセス（`FetchUpdateService`）で行われます。
