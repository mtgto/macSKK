# 一般

設定画面の「一般」タブにある設定項目です。

## Enterキーの動作

**Enterキーで変換候補確定後に改行を入力**

Enterキーで変換候補を確定したときに、続けて改行も送るかどうかを設定します。

## キーボードレイアウト

**キー配列**

ローマ字入力に使うキー配列を選択します。デフォルトはQWERTY配列です。システムで有効な英語用キー配列のみ表示されます（DvorakやColemakなどに変更可能）。

## 変換候補リストの方向

**変換候補リストの表示方向**

変換候補パネルの表示方向を縦・横から選択します。

## 候補パネルに一度に表示する変換候補の数

**変換候補パネルに表示する変換候補の数**

変換候補パネルに同時に表示する候補数（1〜9）を設定します。

## 注釈

**現在の変換候補の注釈を表示**

変換候補の近くに注釈（辞書の説明文）を表示するかどうかを設定します。

**注釈に使用するシステム辞書**

注釈に使用するシステム辞書を「スーパー大辞林」「ウィズダム英和・和英」から選択します。

## 変換候補から確定するキー

**変換候補の決定に使用するキー**

変換候補一覧から候補を確定するショートカットキーを選択します。

| 選択肢 | キー |
|---|---|
| `123456789` | 数字キー（デフォルト） |
| `ASDFGHJKL` | ホームポジション（QWERTY） |
| `AOEUIDHTN` | ホームポジション（Dvorak） |

## 句読点の設定

カンマ・ピリオドキーを押したときに入力される文字を設定します。

**カンマの挙動**

| 選択肢 | 入力される文字 |
|---|---|
| ローマ字かな変換ルールに従う | ローマ字かな変換ルールの設定に従う（デフォルト） |
| "、" を入力 / Enter "、" | 読点 `、` |
| "，" を入力 / Enter "，" | 全角カンマ `，` |

**ピリオドの挙動**

| 選択肢 | 入力される文字 |
|---|---|
| ローマ字かな変換ルールに従う | ローマ字かな変換ルールの設定に従う（デフォルト） |
| "。" を入力 | 句点 `。` |
| "．" を入力 | 全角ピリオド `．` |

## カタカナ確定時に辞書登録

**カタカナで確定した単語を辞書に登録する**

カタカナで変換を確定した場合、その結果をユーザー辞書に登録するかどうかを設定します。

## 単語登録中の動作

**単語登録中に先頭スペースを無視する**

単語登録モードで入力テキストの先頭にあるスペースを無視するかどうかを設定します。

**単語登録中に前候補キーもしくはバックスペースキーで候補選択に戻る**

単語登録モード中に登録テキストが空の状態で前候補キーまたはバックスペースを押したとき、単語登録を中断して候補選択に戻るかどうかを設定します。

**単語登録中にTabで読みを入力する**

単語登録モード中にTabキーで読みを補完するかどうかを設定します。

## 入力モードの表示

**入力モードアイコンを表示**

入力モードが変わったときにモーダルで表示するかどうかを設定します。

**▽と▼を表示する**

変換開始位置を示す ▽（読み入力中）や ▼（候補選択中）マーカーを表示するかどうかを設定します。

**入力モードの色をカスタマイズ…**

入力モードモーダルの背景色・文字色をカスタマイズするシートを開きます。

## インライン候補表示

**インラインで表示する変換候補の数**

変換候補をパネルではなくインライン（テキスト入力位置の近く）で表示する候補数を設定します（0にするとインライン表示なし）。

## 候補選択時のバックスペース

**変換候補選択中のバックスペース**

変換候補を選択中にバックスペースを押したときの動作を設定します。

| 選択肢 | 動作 |
|---|---|
| 一つ前の状態に戻す | インライン表示中または変換候補リストの1ページ目のときはキャンセルして変換開始前に戻す。2ページ目以降では前のページに戻す（デフォルト） |
| インライン時は一文字削除して確定 | インライン表示中は末尾1文字を削除して確定する。変換候補リスト表示中は前のページに戻す（ddskk・AquaSKK互換） |
| 常に一文字削除して確定 | インライン・変換候補リスト表示を問わず末尾1文字を削除して確定する（skkeleton互換） |
| 前候補に戻す | 前の変換候補に移動する |

## プライベートモード

**プライベートモードではユーザー辞書を参照しない**

[プライベートモード](../features/private-mode)が有効な場合に、ユーザー辞書も参照しないようにするかどうかを設定します。
