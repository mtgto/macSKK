# 変換候補パネル

設定画面の「変換候補パネル」タブにある設定項目です。変換候補と注釈のフォント・背景色を設定します。

## 変換候補 {#candidates}

**フォント名**

変換候補パネルに使うフォントを選択します。「デフォルト」はシステムのデフォルトフォントを使用します。

**変換候補のフォントサイズ**

変換候補のフォントサイズを6〜30ptの範囲で選択します（デフォルト: 13）。

**背景色を上書きする**

変換候補パネルの背景色をカスタムカラーで上書きするかどうかを設定します。

**背景色**

上書き設定が有効な場合に使用する背景色を選択します。

## 注釈 {#annotation}

**フォント名**

注釈テキストに使うフォントを選択します。

**注釈のフォントサイズ**

注釈のフォントサイズを6〜30ptの範囲で選択します（デフォルト: 13）。

**背景色を上書きする**

注釈の背景色をカスタムカラーで上書きするかどうかを設定します。

**背景色**

上書き設定が有効な場合に使用する背景色を選択します。

::: tip
注釈を表示するかどうかは[一般設定](./general#注釈)で設定します。
:::
