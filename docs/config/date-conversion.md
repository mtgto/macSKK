# 日付変換

設定画面の「日付変換」タブにある設定項目です。

SKKの数値変換機能を使って日付を入力できます。「きょう」などの読みを変換すると、設定したフォーマットで今日の日付が変換候補に表示されます。

## 読み

変換のトリガーとなる読みを管理します。デフォルトでは「きょう」「yesterday」などが登録されています。

- **追加**: `+` ボタンで読みと基準日（今日・昨日・明日など）を追加します
- **削除**: 読みを選択して `-` ボタンで削除します
- **並び替え**: ドラッグで順序を変更できます
- **編集**: 読みをダブルクリックで編集シートを開きます

## 変換候補

変換結果のフォーマットを管理します。

- **追加**: `+` ボタンでフォーマット・ロケール・カレンダーを設定して追加します
- **削除**: 候補を選択して `-` ボタンで削除します
- **並び替え**: ドラッグで変換候補の表示順を変更できます
- **編集**: 候補をダブルクリックで編集シートを開きます

フォーマットは `DateFormatter` の書式文字列を使用します（例: `yyyy年M月d日`、`M/d/yyyy`）。ロケールとカレンダー（グレゴリオ暦・和暦など）も設定可能です。
