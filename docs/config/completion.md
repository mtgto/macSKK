# 補完

設定画面の「補完」タブにある設定項目です。

読み入力中に変換候補や読みの補完を表示する機能の設定です。

## 補完候補を表示 {#show-completion}

補完候補の表示を有効にするかどうかを設定します。オフにすると以下の設定も無効になります。

## 変換候補を補完候補として表示 {#show-candidate-for-completion}

読み入力中に、補完候補に対する変換候補を表示するかどうかを設定します。

## 最初の補完候補をピリオドキーで確定する {#confirm-the-first-completion-by-period}

**変換候補を補完候補として表示**が有効な場合に、ピリオドキーで最初の補完候補を確定するかどうかを設定します。

## 読み入力から候補選択に切り替わるまでの時間 {#completion-confirmation-time-limit}

**変換候補を補完候補として表示**が有効な場合に、読み入力を開始してから変換候補選択モードに自動的に切り替わるまでの時間を設定します。100ms〜1000ms（100ms刻み）または「24時間」から選択します。

## ユーザー辞書だけでなくすべての辞書から補完を探す {#find-completion-from-all-dictionaries}

補完候補の検索対象をユーザー辞書だけでなく、有効なすべての辞書に広げるかどうかを設定します。

::: tip 補完の動作について
読みの補完機能の詳細は[単語登録・補完](../features/word-registration#読みの補完)を参照してください。
:::
