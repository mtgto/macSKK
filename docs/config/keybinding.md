# キーバインド

設定画面の「キーバインド」タブにある設定項目です。

`q`・`l`・`Ctrl-j` などSKKで使用するキーバインドを変更できます。

## キーバインドセットの選択

**設定**

Pickerでキーバインドセットを選択します。デフォルトセットは編集・削除できません。

## セットの管理

セット名の右にある `…` メニューから操作できます。

| 操作 | 日本語
|---|---|
| リネーム | セット名を変更する（デフォルトセットは不可） |
| 複製 | 選択中のセットを複製する |
| 削除 | セットを削除する（デフォルトセットは不可） |

カスタマイズするには、まずデフォルトセットを**複製**してから編集してください。

## アクションとキーの一覧

アクション名と割り当てられたキーの一覧テーブルが表示されます。デフォルトから変更されているアクションは**太字**で表示されます。

テーブルの行をダブルクリックするか、右クリックメニューの「**編集**」からキーを変更できます。「**リセット**」でそのアクションのキーをデフォルトに戻せます。

### カスタマイズできるアクション一覧

| アクション名 | デフォルトキー | 説明 |
|---|---|---|
| ひらがなモード | ⌃J | ひらがな入力モードに切り替える |
| かなカナモード切り替え | Q | ひらがな⇔カタカナモードを切り替える（通常・候補選択時） |
| 入力中のかなカナ切り替え | Q | 入力中の未確定文字をひらがな⇔カタカナに変換して確定する（読み入力中のみ） |
| 半角カナモード | ⌃Q | 半角カナ入力モードに切り替える |
| 直接入力 | L | 半角英数（直接入力）モードに切り替える |
| 全角英数モード | ⇧L | 全角英数モードに切り替える |
| Abbrevモード | / | 英字で読みを入力するAbbrevモードに入る |
| 直接入力時Abbrevモード | ⌃; | 直接入力モードからAbbrevモードに入る |
| 未確定入力開始 | ⇧Q | 未確定文字列があれば確定してから新たに未確定入力を開始する |
| Sticky Shift | ; | 次の1文字だけShiftを有効にする（送り仮名など） |
| Enter | Return | 確定・改行 |
| Space | Space | 変換・スペース入力 |
| Shift-Space | ⇧Space | 補完候補がある場合はその候補で変換を開始する |
| 前の変換候補を選択 | X | 前の変換候補に戻る |
| Tab | Tab | 読みの補完候補を確定する |
| Backspace | BS, ⌃H | カーソル前の文字を削除する |
| Delete | Del, ⌃D | カーソル後の文字を削除する |
| キャンセル | Esc, ⌃G | 未確定入力や単語登録をキャンセルする |
| 左 | ←, ⌃B | カーソルを左に移動する |
| 右 | →, ⌃F | カーソルを右に移動する |
| 下 | ↓, ⌃N | カーソルを下に移動する |
| 上 | ↑, ⌃P | カーソルを上に移動する |
| 行頭へ移動 | ⌃A | カーソルを行の先頭に移動する |
| 行末へ移動 | ⌃E | カーソルを行の末尾に移動する |
| 選択中の変換候補の削除 | ⇧X | 選択中の変換候補をユーザー辞書から削除する |
| 登録モードでペースト | ⌃Y | 単語登録モード中にクリップボードからテキストをペーストする |
| 選択テキストを再変換 | ⌃/ | 選択中のテキストを辞書から逆引きして再変換する |
| 接頭辞/接尾辞 | ⇧. (>) | 接頭辞・接尾辞の入力モードに入る |
| 英数キー | 英数 | 直接入力モードに切り替える（JIS配列の英数キー） |
| かなキー | かな | ひらがなモードに切り替える（JIS配列のかなキー） |

::: tip
おかしな挙動や設定にない特殊なキーバインドを希望する場合は [Issue](https://github.com/mtgto/macSKK/issues) でお知らせください。
:::
