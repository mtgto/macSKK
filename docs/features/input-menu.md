# 入力メニュー

macSKKが入力ソースとして選択されているとき、macOSメニューバー右側の**入力メニュー**からさまざまな操作を行えます。

## メニュー項目

### 設定… {#preferences}

設定画面を開きます。

### ユーザー辞書を今すぐ保存 {#save-user-dictionary}

ユーザー辞書をすぐにファイルに書き出します。未保存の変更がある場合のみ表示されます。

通常はバックグラウンドで自動保存されますが、即時保存したい場合に使用します。詳細は[辞書](../config/dictionary)を参照してください。

### 直近の辞書登録を取り消す {#undo-recent-dictionary-registrations}

直近に登録した単語をユーザー辞書から削除できます。プライベートモードが無効かつ直近の登録がある場合のみ表示されます。

### プライベートモード {#private-mode}

プライベートモードの有効・無効を切り替えます。有効時はチェックマークが付きます。

詳細は[プライベートモード](./private-mode)を参照してください。

### "(アプリ名)"では直接入力 {#direct-input-from}

最前面のアプリでの直接入力モードの有効・無効を切り替えます。有効時はチェックマークが付きます。Bundle Identifierを持たないアプリでは表示されません。

詳細は[直接入力](../config/direct-mode)を参照してください。

### 空文字挿入 (互換性) {#insert-blank-string-for-workaround}

最前面のアプリに対して「空文字挿入」ワークアラウンドの有効・無効を切り替えます。有効時はチェックマークが付きます。

### 1文字目を未確定扱い (互換性) {#treat-first-char-as-marked-for-workaround}

最前面のアプリに対して「1文字目を常に未確定扱い」ワークアラウンドの有効・無効を切り替えます。有効時はチェックマークが付きます。

### 空のときには▽▼を表示 (互換性) {#show-marker-when-empty-for-workaround}

最前面のアプリに対して「空のときには▽▼を表示」ワークアラウンドの有効・無効を切り替えます。有効時はチェックマークが付きます。

上記3つのワークアラウンドの詳細は[互換性の設定](../config/workaround)を参照してください。

### SKKServを利用する {#use-skkserv}

SKKServの有効・無効を切り替えます。有効時はチェックマークが付きます。

詳細は[辞書](../config/dictionary#skkserv)を参照してください。
