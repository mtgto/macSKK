# 単語登録

## 単語登録モード

有効な辞書に変換結果が見つからない場合、自動的に単語登録モードへ移行します。

例: "あああ" を変換しようとして辞書にない場合 → `[登録：あああ]` と表示されます。

テキストを入力してEnterを押すと、ユーザー辞書にその読みで登録されます。単語登録モード中にさらに変換しようとした際に候補がない場合は、入れ子で単語登録モードに入ることはできません。

**ペースト**: 単語登録モード中は `C-y` でクリップボードからペーストできます (通常の `Cmd-v` はアプリに取られるため使えません)。

**キャンセル**: `Esc` または `C-g` でキャンセルします。

**直前に登録した単語を削除**: 誤って登録してしまった単語を削除したい場合は入力メニューから削除することが可能です。

## ユーザー辞書から単語を削除

変換候補が選択されている状態で `Shift-x` を入力すると、削除確認画面に切り替わります。

```
(よみ) /(変換結果)/ を削除します(yes/no)
```

`yes` とEnterで削除、`no` または `Esc`/`C-g` でキャンセルします。

現在はユーザー辞書のエントリのみ削除対象です。

## 読みの補完

読みを入力中に、ユーザー辞書の送りなしエントリから先頭が一致する変換履歴がある場合、入力テキストの下部に候補が表示されます。`Tab` キーで表示されている読みまで補完されます。

補完の対象はユーザー辞書の送りなしエントリです。

## ユーザー辞書の自動保存

変換・単語登録でユーザー辞書が更新された場合、一定期間おきに自動保存されます。macSKKが正常終了する際、未保存の更新があれば保存してから終了します。

即時保存したい場合は **入力メニュー → ユーザー辞書を今すぐ保存** を選択してください。

::: warning
`Command + Option + Esc` による強制終了やシグナルによる終了では保存されない場合があります。
:::
